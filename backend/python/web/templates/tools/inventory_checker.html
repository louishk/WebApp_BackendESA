How to Use

click to toggle
  1. Select sites from the dropdown and click Load Units.
  2. Set Type Mappings first (shared across all sites) — go to Settings > Type Mappings to assign a Type Code and Climate Code to each source type.
  3. Review units row by row. The filter defaults to Unreviewed so you only see units that still need attention.
  4. For each unit, verify or adjust the Type, Climate, Size, Range, Shape, Pillar dropdowns. Changes with a yellow background are per-unit overrides.
  5. Tick the checkbox in the first column to mark a unit as reviewed — this saves automatically.
  6. When you're done adjusting overrides, click Save Overrides to persist all dropdown changes.
  7. Use the filter dropdown to switch between All / Unreviewed / Reviewed to track your progress.
  8. Export to Excel at any time for an offline copy (includes a Reviewed column).
