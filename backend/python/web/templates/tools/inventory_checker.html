How to Use

click to toggle
  1. Select sites from the dropdown and click Load Units.
  2. Set Type Mappings first (shared across all sites) — go to Settings > Type Mappings to assign a Type Code and Climate Code to each source type.
  3. Filter columns using the dropdowns in the header row to narrow down units.
  4. Select units with checkboxes (or Select All) and use the Batch Actions bar to set Type, Climate, Shape, or Pillar for all selected units at once.
  5. For individual units, adjust the Type, Climate, Size, Range, Shape, Pillar dropdowns directly in the row.
  6. Tick the review checkbox in the first column to mark a unit as reviewed — this saves automatically.
  7. Click Save Overrides to persist all dropdown changes.
  8. Export to Excel at any time for an offline copy.
