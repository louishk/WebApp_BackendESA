Billing Mode Reference

Anniversary billing: tenant pays full month from move-in date each cycle. No proration.
1st-of-month billing:

  • Day 1 → full month rent
  • Day 2 to X → prorated current month
  • Day X+1 to end-of-month → prorated current + full next month
Where X = iDayStrtProratePlusNext.
