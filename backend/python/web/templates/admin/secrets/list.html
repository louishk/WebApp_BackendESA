Bootstrap Secrets (env vars only)

These must be set as environment variables and cannot be stored in the DB vault:

  • DB_PASSWORD - Backend database (bootstrap)
  • PBI_DB_PASSWORD - PBI database (bootstrap)
  • VAULT_MASTER_KEY - Encryption key (bootstrap)

Expected Secrets

These secrets are referenced in the configuration files:

  • MS_OAUTH_CLIENT_SECRET - Microsoft OAuth
  • JWT_SECRET - JWT signing key
  • FLASK_SECRET_KEY - Flask sessions
  • SOAP_API_KEY - SMD SOAP API
  • SOAP_CORP_PASSWORD - SMD SOAP password
  • SUGARCRM_PASSWORD - SugarCRM
  • SLACK_WEBHOOK_URL - Slack alerts
  • VM_SSH_PASSWORD - Azure VM
